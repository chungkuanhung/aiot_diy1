根據影片內容，這份作業的主要要求與步驟如下：

**1. 工具安裝與環境設置**

* **下載與安裝 IDE**：下載並安裝 Google 出品的「Anti Gravity」IDE 工具 [[00:19](http://www.youtube.com/watch?v=bYoq3pbOlNM&t=19)]。
* **登入帳號**：需登入 Google 帳號，以便取得授權使用該 IDE 內建的 AI 大語言模型 (Agent) 功能 [[01:21](http://www.youtube.com/watch?v=bYoq3pbOlNM&t=81)]。
* **建立專案資料夾**：在電腦中建立並開啟一個新的資料夾（例如命名為 `lecture one`）作為專案目錄 [[01:47](http://www.youtube.com/watch?v=bYoq3pbOlNM&t=107)]。

**2. 製作個人網頁 (Personal Web Page)**

* **生成單頁網頁**：利用 AI Agent 下達指令，請它協助撰寫一個單頁式的個人網頁 (Single page personal web page) [[06:52](http://www.youtube.com/watch?v=bYoq3pbOlNM&t=412)]。
* **必備內容**：網頁中必須包含**你自己的名字**以及**目前的時間 (Current Time)** [[05:48](http://www.youtube.com/watch?v=bYoq3pbOlNM&t=348)]。
* **版面設計**：老師建議可以尋找自己喜歡的網頁模板 (Template) 讓 Agent 參考製作，盡量不要跟老師的陽春範例一模一樣 [[06:06](http://www.youtube.com/watch?v=bYoq3pbOlNM&t=366)]。
* **響應式設計 (RWD)**：要求 Agent 修改程式碼，使網頁具備響應式設計 (Responsive design with RWD style)，讓網頁能適應不同螢幕大小 [[17:41](http://www.youtube.com/watch?v=bYoq3pbOlNM&t=1061)]。

**3. 上傳至 GitHub 並進行部署**

* **建立 Repository**：在 GitHub 上建立一個全新的 Repository（影片中示範命名如 `0225 DIC` 等） [[10:56](http://www.youtube.com/watch?v=bYoq3pbOlNM&t=656)]。
* **推上 GitHub**：將 GitHub Repository 網址貼給 AI Agent，請 Agent 自動幫忙將所有檔案 push 到 GitHub 上 [[11:31](http://www.youtube.com/watch?v=bYoq3pbOlNM&t=691)]。
* **部署 GitHub Pages**：前往 GitHub Repository 的設定 (Settings)，找到 Pages 的選項，並選擇從 `main` 分支部署，以產生對外的網頁連結 [[15:53](http://www.youtube.com/watch?v=bYoq3pbOlNM&t=953)]。

**4. 完善 README 文件**

* **產生 Summary**：請 Agent 自動產生一份網頁專案的摘要，並存檔為 `README.md` [[14:39](http://www.youtube.com/watch?v=bYoq3pbOlNM&t=879)]。
* **加入截圖與網址**：在 README 中放上個人網頁的首頁截圖 (Snapshot) [[19:57](http://www.youtube.com/watch?v=bYoq3pbOlNM&t=1197)]，並將剛才部署成功的 Live Demo 網址也附加上去 [[19:34](http://www.youtube.com/watch?v=bYoq3pbOlNM&t=1174)]。
* **最終上傳**：確認這些細節與修改都完成後，再次將所有的更新推送 (Push everything) 至 GitHub [[20:33](http://www.youtube.com/watch?v=bYoq3pbOlNM&t=1233)]。
